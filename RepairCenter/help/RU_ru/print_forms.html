РЕДАКТИРОВАНИЕ ПЕЧАТНЫХ ФОРМ

RepairCenter использует для вывода на печать библиотеку QtRPT. Для создания шаблонов печатных форм используется программа QtRptDesigner.

Краткая справка по QtRptDesigner. Полную документацию можно посмотреть на сайте qtrpt.sourceforge.net.

Квитанция

Печатается при добавлении нового заказа.

Имя файла — ./print_forms/order.xml

Доступные поля:

Тег:

Поле в экранной форме:

[number]

Номер заказа

[date]

Дата

[customer]

Имя клиента

[phone]

Номер телефона клиента

[product_type]

Тип изделия

[product]

Изделие

[serial]

Серийный номер

[disease]

Неисправность

[cond]

Состояние

[complect]

Комплектность

[cost]

Примерная стоимость

[acceptor]

Имя приёмщика
Акт диагностики

Печатается при выдаче заказ без ремонта.

Имя файла — ./print_forms/act_diag.xml

Доступные поля:

Тег:

Поле в экранной форме:

[number]

Номер заказа

[date]

Дата

[customer]

Имя клиента

[product_type]

Тип изделия

[product]

Изделие

[serial]

Серийный номер

[inspect]

Результаты осмотра

[defects]

Выявленные неисправности

[recomm]

Рекомендации к ремонту

[giver]

Имя приёмщика
Акт выполненных работ и Гарантийный талон

Печатается при выдаче заказа с ремонтом.

Имя файла — ./print_forms/act_works.xml, ./print_forms/warranty_works.xml

Доступные поля:

Тег:

Поле в экранной форме:

[number]

Номер заказа

[date]

Дата

[customer]

Имя клиента

[product_type]

Тип изделия

[product]

Изделие

[serial]

Серийный номер

[giver]

Имя приёмщика

[warranty]

Состояние

[NN]

Номер строки таблицы (обязательный тег)

[work]

Наименование работ (только в таблице)

[quant]

Количество работ (только в таблице)

[summ]

Сумма стоимости работ и запчастей (только в таблице)



Стоимость работ рассчитывается по формуле: стоимостьРаботы=(суммаРаботИЗапчастей)/количествоРабот. Расчёт стоимости и общей суммы следует производить средствами QtRpt.
