Установка в Linux

Для сервера установите любым удобным способом следующие пакеты и их зависимости (на примере репозитория Ubuntu 14.04 и Qt5): mysql-server, libqt5sql5mysql, libqt5script5, libqt5help5, libqt5xml5, libqt5printsupport5.

Установка клиента производится аналогично, с той разницей, что mysql-server устанавливать не нужно, только вписать настройки.

Создание базы данных

Запустите DBTool, выберите вкладку «Создать БД», введите пароль пользователя «root», указанный при установке MySQL, желаемое название БД, имя и пароль нового пользователя БД и нажмите «Создать». При выборе опции «Сохранить настройки» будет создан файл «settings.conf» с настройками подключения к созданной БД. После этого можно запускать RepairCenter и начинать работу .
