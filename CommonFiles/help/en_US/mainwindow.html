The option "Notify" displays orders where field "Customer notified" = 0.
Given orders are only displayed by pressing the button "All", and click on this button resets any filters.


2. By order number or customer name.

Select an option from the list and start typing. You must enter the full number of the order, or any part of the customer's behalf.


3. By date range.

Select the beginning and end of the range and click the binoculars icon. Filtering is performed only by the date of order reception.


Filters do not add up, the choice is available only for one parameter.

You can also sort the results in ascending and descending order by clicking on the column headers.

When you select a row in the table in the quick view area displays a summary of the order.


Editing order state



For quick editing in the main window available fields "State", "Repairer", "Customer notified" and "Comment", after their changes you need to click "Save".

For full access to order information buttons "View" and "Edit" available. The order number and the customer name remains unchanged and can not be edited.
