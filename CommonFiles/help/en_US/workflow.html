WORK WITH ORDERS

Creating a new order

To place new order, you can click on the main toolbar or menu item in "Actions". This creates a new empty record in the database and a new window opens with the established current date and order number.


In the "Product type" field, corresponding catalogue contents is displayed. You can also enter new text and click the button to the right to add a new entry in the directory. If you do not press the Add button, the current text will be printed, but the type of product will not be saved in the database.


You can immediately appoint a responsible repairer, to mark the order available for all masters just don't touch this field.


Diagnosis report

To create a new report, you can choose the order in a table in the main window and click the appropriate button or menu item. This will open a form with the filled "Order #". If the order line is not selected, the order number will need to be filled manually. Text in each field is limited to 1024 characters including spaces. For each order can be only one report of diagnosis.




Work report

Creates similarly to the diagnostics report. For each type of work you need to create a separate report. Also, it allows multiple masters to work on order, with a separate account of salaries. You can create an unlimited number of reports from different masters. You can add works and spare parts to catalogues. The number of parts is editable in the table.





Issuance without repair

Creating similar to the previous one. If you change the field "Order #" form is populated with corresponding report data. You can edit the text for print, but changes will not be saved. When saving the order status changes to "Given with diagnosis."





Issuance wth repair

Creating similar to the previous one. If you change the field "Order #" form is populated with corresponding reports data. The text from the "Warranty" is printed on the warranty card. In the tables, you can edit the data for printing, but the changes will not be saved. Button to the right of the field "Amount" recalculates the total amount of tables. When saving the order status changes to "Given with repair."



When printing cost of the each work is calculated as follows: Costs = ( SumOfWork + SumOfParts) / NumberOfWorks. Price calculation and the total amount should be made by QtRpt.
