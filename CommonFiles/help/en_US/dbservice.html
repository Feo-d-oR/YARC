DATABASE SERVICE

DB update

When upgrading to a new version of the application, you must update the database structure. To do this, run DBTool, select the tab "Update DB" and enter the current connection settings.



DB Cleanup

Tab "Cleaning" is necessary to optimize and reduce the response time of the database.


"Clean salaries" - deletes all data on the calculated and paid wages. Useful at weak server from past century, since all calculations are performed by MySQL.

"Clean completed orders" - deletes records of given orders, but leaves the customers data. Salary is also not affected.
