CATALOGS

All catalogs are edited on the same principle: "Add" button inserts a new record without affecting the database; "Save" button writes all changes to the database; "Remove" button cancels any unsaved changes, deletes the selected entry from the database and updates the table; "Close" button discards all unsaved changes and closes the window. Editing is performed in the table itself, or the side panel.
