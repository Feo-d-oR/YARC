ПРОЧИЕ ДЕЙСТВИЯ

Печать рукописных документов

На случай отключения электричества и выездов.

Печать производится на принтер по умолчанию, без предварительного просмотра. Для квитанций резарвируются записи в базе данных (в целях сохранения единой нумерации).

Значение «reserved» в поле «Изделие» исполользуется в обработчике редактирования заказа, меняется только при вносе всех данных.

Файлы печатных форм:

Квитанция - order_hw.xml

Акт диагностики - act_diag_hw.xml

Акт выполненных работ - act_works_hw.xml

Гарантийный талон - warranty_works_hw.xml
