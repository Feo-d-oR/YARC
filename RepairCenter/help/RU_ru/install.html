Установка в Linux

Для сервера установите любым удобным способом следующие пакеты и их зависимости (на примере репозитория Ubuntu 14.04 и Qt5): mysql-server, libqt5sql5mysql, libqt5script5, libqt5help5, libqt5xml5, libqt5printsupport5. Также нужно скопировать содержимое папки «lib» в /usr/lib.

Установка клиента производится аналогично, с той разницей, что mysql-server устанавливать не нужно, только вписать настройки.

Создание базы данных

Запустите RepairCenter, в предложении создать базу данных нажмите «Создать». Введите пароль пользователя «root», указанный при установке, желаемое название БД, имя и пароль нового пользователя БД и нажмите «Создать и закрыть». При выборе опции «Сохранить настройки» будет создан файл «settings.conf» с настройками подключения к созданной БД. После этого перезапустите приложение.

На данный момент не производится обработка ошибок, для проверки можно попытаться создать БД с таким же именем, в случае успеха появится сообщение «База данных с таким именем уже существует». Также можно использовать сторонние средства администрирования MySQL.
