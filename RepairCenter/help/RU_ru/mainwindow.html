Главное окно приложения разделено на две области: главная таблица заказов с поиском и быстрый просмотр.



Поиск и сортировка

Выданные заказы отображаются только по нажатию на кнопку «Все», также нажатие на эту кнопку сбрасывает любые фильтры.


2. По номеру заказа или имени клиента.

Выберите вариант из списка и начните ввод. Нужно ввести полностью номер заказа, или любую часть имени клиента.


3. По диапазону дат.

Выберите начало и конец диапазона и нажмите кнопку с биноклем. Фильтрация осуществляется только по дате приёма заказа.


Фильтры не складываются, доступен выбор только по одному параметру.

Также можно сортировать результаты по возрастанию и убыванию, нажимая на заголовки столбцов.

При выделении строки таблицы в правой панели отображается краткая информация по заказу.


Редактирование состояния заказа.


Для быстрого редактирования в главном окне доступны поля «Статус», «Мастер» и «Комментарий», после их изменения нужно нажать кнопку «Сохранить».

Для полного доступа к информации о заказе доступны кнопки «Просмотр» и «Изменить». Номер заказа и клиент остаются неизменными и недоступны для редактирования
