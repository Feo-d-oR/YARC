РАБОТА С ЗАКАЗАМИ

Создание нового заказа

Добавить заказ можно нажав кнопку на главной панели или пункт в меню «Действия». При этом создаётся новая пустая запись в базе данных и открывается новое окно с установленными текущей датой и номером заказа.

В поле «Тип изделия» выводится соответствующий справочник. Также можно ввести новый текст и нажать кнопку справа, чтобы добавить новый элемент в справочник. Если не нажимать кнопку добавления, текущий текст будет выведен на печать, но тип изделия нигде в базе не сохранится.

Можно сразу назначить ответственного мастера, для пометки заказа доступным для всех мастеров следует создать отдельного сотрудника с пустым именем.



Отчёт о диагностике

Создать новый отчёт можно выбрав заказ в таблице в главном окне и нажав соответствующую кнопку или пункт меню. При этом откроется форма с заполненным полем «Номер заказа». Если строка заказа не выделена, или при добавлении из журнала отчётов номер заказа нужно будет вести вручную. Ввод текста в каждом поле ограничен 1024 символами включая пробелы. Для каждого заказа может быть только один отчёт диагностики.



Отчёт о работе

Создание аналогично отчёту диагностики. Для каждого вида работы нужно создавать отдельный отчёт. Можно создавать неограниченное количество отчётов от разных мастеров. Можно добавлять работы и запчасти в справочники. Количество запчастей редактируется в таблице.



Выдача без ремонта

Создание аналогично предыдущим. При изменении поля «Номер заказа» форма заполняется данными соответствующего отчёта. Можно редактировать текст для печати, но сохранены изменения не будут. При сохранении статус заказа меняется на «Выдан с актом диагностики».



Выдача с ремонтом

Создание аналогично предыдущим. При изменении поля «Номер заказа» форма заполняется данными соответствующих отчётов. Текст из поля «Гарантия» печатается в гарантийном талоне. В таблицах можно редактировать данные для печати, но изменения сохранены не будут. Кнопка справа от поля «Сумма» пересчитывает общую сумму по таблицам.

При выводе на печать стоимость работ рассчитывается по формуле: стоимостьРаботы=(суммаРабот+суммаЗапастей)/количествоРабот. Расчёт стоимости и общей суммы следует производить средствами QtRpt.
