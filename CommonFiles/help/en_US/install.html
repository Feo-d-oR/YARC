Installation under Linux

For server install in any convenient way followinf packages and their dependencies: (for example, Ubuntu 14.04 repository and Qt5): mysql-server, libqt5sql5mysql, libqt5script5, libqt5help5, libqt5xml5, libqt5printsupport5.

Client installation is performed similarly, with the difference that the mysql-server installation isn't required, just enter the settings.


Database creation

Start DBTool, select the tab "Create DB", enter the password for the user «root», specified when installing MySQL, the name of the desired database, name and password for the new database user and click "Create". If you select "Save settings", the «settings.conf» file will be created and filled with settings for created database. You can then run RepairCenter and start working.
